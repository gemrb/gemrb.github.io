---
title: SetFullScreen
module: GemRB
---

**Prototype:** GemRB.SetFullScreen (flag)

**Description:** Adjusts fullscreen mode.

**Parameters:**
  * flag:
    * -1 -  toggle fullscreen mode
    * 0 - set windowed mode
    * 1 - set fullscreen mode

**Return value:** N/A

**See also:** [SetGamma](SetGamma.md)
