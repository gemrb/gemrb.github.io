module: _GemRB

**Metaclass Prototype:** EnableBorder (BorderIndex, enabled)

**Description:** Enable or disable specified button border/frame/overlay.

**Parameters:** 
  * BorderIndex - 0, 1 or 2
  * enabled - boolean, true enables the border

**Return value:** N/A
